# Sentiment Analysis of Movie Reviews by Genre

## Overview

This project explores how audience sentiment varies across different movie genres using Natural Language Processing (NLP) techniques. By analyzing more than 46,000 movie reviews, the project investigates whether certain genres receive more positive or negative reactions from audiences and how emotional responses differ across genres.

The analysis was performed using two popular sentiment analysis techniques: **VADER** and **TextBlob**. Sentiment scores generated by these methods were used to compare audience opinions across genres such as Comedy, Drama, Horror, Romance, Adventure, Thriller, and others.

## Objectives

The main goals of this project were to:

* Analyze movie reviews and identify audience sentiment.
* Compare sentiment across different movie genres.
* Determine which genres receive the most positive and negative responses.
* Explore emotional patterns associated with different genres.
* Visualize sentiment trends using different charts and plots.

## Dataset

The project uses the **Movie Reviews and Emotion Dataset** from Kaggle.

The dataset contains:

* Movie reviews
* User ratings
* Movie genres
* Movie descriptions
* Emotion labels

**Note:** The dataset is not included in this repository due to its large size.

## Methods Used

### Sentiment Analysis

Two sentiment analysis approaches were applied:

* **VADER (Valence Aware Dictionary and Sentiment Reasoner)** for sentiment scoring and classification.
* **TextBlob** for polarity-based sentiment analysis.

### Data Processing

* Genre information was cleaned and transformed for analysis.
* Movies belonging to multiple genres were processed so that each genre could be analyzed independently.
* Sentiment scores were calculated for every review.

### Visualization & Analysis

Several visualizations were created to better understand audience sentiment, including:

* Genre-wise sentiment comparison
* Average sentiment score analysis
* Lollipop plots
* Error bar plots
* Box plots
* Emotion distribution heatmaps

## Key Findings

* Comedy, Adventure, and TV Movie genres generally received the most positive audience sentiment.
* Horror, Thriller, and War genres showed comparatively lower sentiment scores and more negative reactions.
* Different genres exhibit distinct emotional patterns.
* Sentiment analysis can provide meaningful insights into audience preferences and movie reception.
